# StayBnB - A Hotel Booking Website

## Introduction
Welcome to our GitHub repository for StayBnB, an innovative web application designed to revolutionize the way people book accommodations. Inspired by the success of platforms like Airbnb, StayBnB offers a comprehensive solution for travelers and hosts alike. Whether you're looking for a cozy apartment for a weekend getaway or managing your property listings, StayBnB has you covered.

The key features of our website include:
- Login/Signup
- Booking and payment functionality

## Installation & Getting started
- Clone the repository: git clone <a href="https://github.com/tashwini-p/B33-Group1.git">https://github.com/tashwini-p/B33-Group1.git</a>
- Navigate to the project directory on gitbash: cd B33-Group1
- Open the <a href="index.html">index.html</a> file in your preferred web browser from the root directory.

## Usage
<strong> Home Page </strong>
<ul>
   <li>Visit the StayBnB Website. </li>
   <li>Explore the homepage for hotel listings in the most exciting destinations </li>
   <li>Navigate to specific sections like Stays, About Us, Support sections </li>
   <li>Book your hotels by clicking on the "Book Now" button on any of the hotel listings. </li>
</ul>
<img width="939" alt="image" src="https://github.com/tashwini-p/B33-Group1/assets/154405564/e2ec967b-0abd-47a3-88df-7a6753afe32d">
<br> <br>
<strong> Login/SignUp Page </strong>
 <ul>
        <li>Users can log in using a standard email and password combination after registering on the sign up form.</li>
        <li>Account Creation: New users can create an account by providing their email and password.</li>
        <li>Admin Page Integration: With the right credentials for accessing admin rights, users can access the admin dashboard for additional functionalities </li>
</ul>

<img width="952" alt="image" src="https://github.com/tashwini-p/B33-Group1/assets/154405564/181e8fd8-c0c3-48d9-abb2-858a1345f16a">
<br> <br>

<strong> Booking and Payments Page </strong>
<ul>    
   <li> Navigate to the "Stays" Page through the navigation bar.</li>
   <li> Choose a hotel of your liking and click on the "Book Now" button </li>
   <li> The button will redirect you to the payment page.</li>
   <li> Add in all the necessary information and click "confirm" to move on to the payment gateway.</li>
   <li> At the Payment Page, choose a payment method, verify and then click on "Securely Pay"</li>
</ul>

 <strong>Admin Dashboard</strong>
 <ul>
    <li>Navigate to the sign in page and type in the admin credentials</li>
    <li>Explore the dashboard and the statistics visible</li>
    <li>Click on "Our Hotels" in the sidebar to access the editing page where an admin can add more hotel listings, edit or delete existing ones and even filter and sort through the content</li>
    <li>To go back to the landing page, simply click on the home icon on the dashboard navbar.</li>
